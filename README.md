# node-red-contrib-flowgen

Overview

## Description

swagger.json から node-red の flows を生成する

## Requirement

node-red  
swagger-parser  
fs  

## Usage

```
node flowgen.js input output

ex)
node flowgen.js https://petstore.swagger.io/v2/swagger.json /tmp/flows.json
```

## Install

```
npm i
```
